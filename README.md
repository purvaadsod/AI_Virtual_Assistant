# AI Voice Assistant

An AI-powered Voice Assistant built using the MERN Stack that listens to voice commands, understands natural language, executes tasks, and responds using human-like speech. The assistant can answer questions, open websites, provide information, and perform browser-based actions through voice commands.

## Features

* Voice activation using a custom wake word
* Speech-to-text conversion
* AI-powered natural language understanding
* Text-to-speech responses
* Real-time voice interaction
* Open websites through voice commands
* Open YouTube, Google, and other web applications
* Answer general knowledge questions
* Tell the current date and time
* Responsive MERN-based interface

## Tech Stack

### Frontend

* React.js
* JavaScript
* HTML5
* CSS3

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

### AI Integration

* Gemini API

## How It Works

1. The user speaks the assistant's wake word.
2. The assistant begins listening for voice input.
3. Speech is converted into text.
4. Gemini AI processes the user's request.
5. The assistant executes the requested command or answers the query.
6. The response is converted into speech and played back to the user.

## Example Commands

* "Open YouTube"
* "Open Google"
* "What is Artificial Intelligence?"
* "Tell me today's date."
* "Who is the Prime Minister of India?"
* "What is the weather today?"
* "Search React tutorials on YouTube."
